## Issue: Script not executing
**Solution:**  
Ensure Python is installed and use:
```bash
python3 script_name.py
```
## Issue: File not found errors

Solution:
Run scripts from the correct directory so relative paths resolve correctly.

## Issue: Permission denied

Solution:
Check file permissions or run:
```
chmod +x script.py
```
