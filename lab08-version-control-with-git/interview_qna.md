Q1. What is Git used for in software and cybersecurity projects?  
A. Git tracks changes, manages versions, and maintains an audit trail.

Q2. What does git init do?  
A. It initializes a new Git repository.

Q3. Why is git status important?  
A. It shows the state of tracked and untracked files.

Q4. What is the purpose of git add?  
A. It stages files for committing.

Q5. What does a commit represent?  
A. A snapshot of changes with a message.

Q6. Why are branches used?  
A. To isolate development work safely.

Q7. What happens during git merge?  
A. Changes from one branch are combined into another.

Q8. What is the purpose of .gitignore?  
A. It prevents unwanted files from being tracked.

Q9. What does git clone do?  
A. It creates a local copy of a repository.

Q10. Why is Git important for cybersecurity teams?  
A. It ensures secure collaboration, audit trails, and configuration integrity.
