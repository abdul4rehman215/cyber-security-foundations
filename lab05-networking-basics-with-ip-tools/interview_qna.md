Q1. What does the ip addr show command display?  
A. It shows network interfaces, assigned IP addresses, and interface status.

Q2. What is the purpose of the loopback interface lo?  
A. It is used for local system communication and testing.

Q3. How do you identify the default gateway on a Linux system?  
A. By using the command ip route show.

Q4. Why is pinging 127.0.0.1 useful?  
A. It verifies that the local TCP/IP stack is functioning correctly.

Q5. What is the difference between pinging an IP address and a domain name?  
A. Pinging a domain also tests DNS resolution in addition to network connectivity.

Q6. What information does traceroute provide?  
A. It shows the network path packets take to reach a destination.

Q7. Why is traceroute -n sometimes preferred?  
A. It avoids DNS lookups, making results faster and clearer.

Q8. What is the purpose of nslookup and dig?  
A. They are used to query DNS records and troubleshoot name resolution issues.

Q9. How does nc -zv help in security analysis?  
A. It checks whether specific TCP ports are open on a target host.

Q10. Why is listing listening services important in cybersecurity?  
A. It helps identify exposed services that may increase the attack surface.
